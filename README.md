# ios-27-Beta
# iOS 27 Beta Web Simulator  這是一個基於 HTML, CSS 與 JavaScript 開發的網頁版 iOS 模擬器，旨在還原流暢的系統過場動畫與次世代的 UI/UX 設計。  ## ✨ 特色 (Features) - **擬真開機畫面**：帶有 Apple Logo 與進度條的開機動畫。 - **Liquid Glass UI**：純粹的高透明度毛玻璃效果，無多餘色偏。 - **硬體加速動畫**：使用 CSS Transform 與 Cubic-bezier 曲線，確保極致滑順的回彈體驗。 - **滑動解鎖手勢**：支援滑鼠拖曳與手機觸控的真實向上滑動解鎖邏輯。
